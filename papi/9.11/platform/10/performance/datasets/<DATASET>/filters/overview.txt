Resource URL: /platform/<<version=10>>/performance/datasets/<DATASET>/filters
()
    Overview: This resource represents the collection of "filters" selected for
              the dataset specified in the URI. A filter is a selection of
              specific values for each metric in its dataset. These values are
              "filtered" such that a workload will not be considered for
              inclusion in the top metrics combinations of a dataset. For
              instance, a dataset with no filter axes defined will consider all
              possible metric combinations within its metrics, whereas a
              dataset with the metric "username" filtered will only consider
              metric combinations where the username metric matches a
              configured filter.
()
     Methods: GET, POST, DELETE
