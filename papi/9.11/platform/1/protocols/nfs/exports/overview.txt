Method GET: List all NFS exports.
()
URL: GET /platform/<<version=1>>/protocols/nfs/exports

Query arguments:
     sort=<enum> Where <enum> is one of id, paths, description.  The field that
                 will be used for sorting. Default is id.
 resume=<string> Continue returning results from previous call using this token
                 (token should come from the previous call, resume cannot be
                 used with other options).
 limit=<integer> Return no more than this many results at once (see resume).
offset=<integer> The position of the first item returned for a paginated query
                 within the full result set.
    scope=<enum> Where <enum> is one of effective, user.  If specified as
                 effective or not specified, all export fields are shown.  If
                 specified as user, only fields with non-default values are
                 shown.
 check=<boolean> Check for conflicts when listing exports.
      dir=<enum> Where <enum> is one of ASC, DESC.  The direction of the sort.
